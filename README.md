
# hello-java-websocket

WebSocket Java Server &amp; Java Client

# How to Run Server?

	mvn jetty:run

# How to Stop Server?

	Ctrl + C (on Windows Command Prompt)

# How to Run Client?

	pkg.WebSocketClientMain.main()

# How to Run Client from JavaScript?

	http://localhost:8080/hello-websocket/index.jsp
